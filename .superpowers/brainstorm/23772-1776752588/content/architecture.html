关键设计决策
⚡
分类：规则优先 + AI 兜底
GitHub/Stripe webhook 走规则自动打标；CLI 手打内容异步丢给 Claude 分类，结果写回 Event
🔒
Review Gate：周期性阻塞
未完成本周 Review 时，Cockpit 宏观看板锁定，只能看到提示页面
🏗️
Project：系统驱动，非人工维护
通过 repo 名 / tag 规则自动聚合 Events，Project 是动态 SQL View，不是人工创建的容器
📡
本地优先，数据主权
所有数据存 ~/.soloos/soloos.db，server 和 web 均本地运行，无云依赖
