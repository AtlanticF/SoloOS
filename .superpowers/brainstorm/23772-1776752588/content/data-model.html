entries
id           TEXT PK
content     TEXT NOT NULL
source      TEXT -- cli/github/stripe/ext
status      TEXT -- pending/processed
quick_tags  TEXT -- JSON array
created_at  INTEGER -- unix timestamp
原始态、不可变。AI 处理前的裸数据。
events
id            TEXT PK
entry_id     TEXT FK → entries
pillar       TEXT -- INPUT/OUTPUT/...
project_id   TEXT FK? → projects
impact_score INTEGER -- -10 to 10
classifier   TEXT -- rule/api-key/skill
metadata     TEXT -- JSON
occurred_at  INTEGER
created_at   INTEGER
结构化、可修改。系统的核心原子。
projects
id             TEXT PK
name          TEXT NOT NULL
status        TEXT -- active/dormant/done
match_rules   TEXT -- JSON: repo/tag 规则
is_auto       INTEGER -- 0/1
first_event_at INTEGER
last_event_at  INTEGER
created_at    INTEGER
逻辑聚合器，ROI 从 events 实时计算。
reviews
id           TEXT PK
period      TEXT -- weekly/monthly
period_start INTEGER
period_end   INTEGER
snapshot    TEXT -- JSON 快照
reflection  TEXT -- 用户手写
ai_insight  TEXT -- AI 生成
completed_at INTEGER -- NULL = 未完成(锁定)
completed_at = NULL → Review Gate 触发锁定。
1 张关联表：事件跨维度链接
event_links  — 产品灵魂：跨 Pillar 关联
source_event_id  TEXT FK → events
target_event_id  TEXT FK → events
link_type        TEXT  -- caused/related/derived
confidence      REAL  -- AI 置信度 0~1，1=人工确认
created_by      TEXT  -- rule/ai/human
示例：一篇竞品分析 (Input) caused 一个新 Feature (Output) derived 一笔收入 (Financial)
